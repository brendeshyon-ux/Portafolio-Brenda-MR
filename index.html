PROYECTOS

Metro CDMX

MiMetro

El proyecto fue desarrollado utilizando la metodología Design Thinking. El desarrollo del proyecto se estructuró en cinco etapas principales:

  • EMPATIZAR: Realicé un benchmarking de aplicaciones de transporte existentes.
  • DEFINIR: Construí un User Persona y un Empathy Map para definir claramente las necesidades del usuario.
  • IDEAR: Desarrollé el User Flow para estructurar una experiencia clara de la aplicación.
  • PROTOTIPAR: Diseñé wireframes de baja fidelidad para validar la estructura y wireframes de alta fidelidad, apoyados por una UI Style Guide.
  • TESTEAR: Realicé pruebas de usabilidad para evaluar la claridad de la interfaz. Los hallazgos permitieron realizar mejoras antes de presentar el prototipo final.
